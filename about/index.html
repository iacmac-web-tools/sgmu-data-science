Цель стратегического проекта

Создание сетевой университетской кооперации в сфере data science в медицине для развития рынка продуктов науки о данных в системе здравоохранения и междисциплинарной подготовки высококвалифицированных кадров.

Задачи стратегического проекта

  1. Формирование кооперации с вузами, осуществляющими подготовку специалистов по техническим, биологическим и медицинским специальностям
  2. Разработка и создание кросс-программ обучения для специалистов технических специальностей и здравоохранения.
  3. Создание инновационной площадки для формирования, консультаций и продвижения мультидисциплинарных команд на рынке здравоохранения.
  4. Привлечение поставщиков услуг здравоохранения для решения практических задач путем создания продуктов науки о данных в здравоохранении.
  5. Создание банка наборов биомедицинских данных (БД) в централизованной среде для облегчения исследовательской деятельности, включая эпидемиологический анализ, анализ общественного здравоохранения и экономики здравоохранения.
  6. Разработка инструментов для анализа и извлечения знаний из сложных наборов биомедицинских данных.
  7. Обучение сотрудников системы здравоохранения использованию созданных инструментов и наборов данных для решения практических задач

Категории обучающихся

Образовательная часть проекта направлена на обучающихся по программам специалитета, ординатуры, аспирантуры, а также в рамках дополнительного профессионального образования.

Вклад проекта «Сетевая университетская кооперация в сфере data science в медицине»

  • Создание и реализация междисциплинарных сетевых проектов по приоритетным тематикам развития науки, технологии техники в Российской Федерации
  • Получение результатов фундаментального и прикладного характера, конкурентоспособных в системе грантового финансирования и актуальных в научном обществе
  • Разработка и внедрение наукоемких инструментов для анализа и извлечения знаний из сложных наборов биомедицинских данных

Ожидаемые результаты от реализации проекта

  • Публикации
  • Привлечение школьников, студентов, молодых ученых к разработкам
  • Всероссийская и международная кооперация в сфере data science с привлечением специалистов из области медицины и информационных технологий
  • РИД: программы для ЭВМ, базы данных, патенты
  • Проекты технической и разрешительной документации на разработки для биомедицины (технологии, услуги)
